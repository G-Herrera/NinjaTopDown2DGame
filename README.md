# NINJA Top-Down Game

Proyecto de videojuego 2D top-down desarrollado en Unity.

## Mecánicas actuales
- Movimiento del jugador
- Enemy FSM Patrol → Chase → Attack
- Sistema de vida
- Sistema de score
- Sistema de coleccionables
- Victoria y GameOver
- Cámara con freeze
- UI y paneles
